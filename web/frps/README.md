# frps & frpc

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```shell
# frps
npm run serve:s

# frpc
npm run serve:c
```

### Compiles and minifies for production

```shell
# frps
npm run build:s

# frpc
npm run build:c
```

### Lints and fixes files

```
npm run lint
```
